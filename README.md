# PHP Quiz Application

## Overview
This is a simple quiz application built using PHP.

## Features
- Multiple-choice questions
- Score calculation
- Instant feedback

## Technologies Used
- PHP
- HTML
- CSS

## How to Run
1. Install XAMPP or WAMP
2. Copy project folder into htdocs
3. Start Apache server
4. Open browser and go to:
   http://localhost/quiz-app
